# Goal

## Project
sanic — a python project.

## Description
An asynchronous web server and framework for Python built on asyncio. It provides fast HTTP request handling with async/await syntax, flexible routing with path parameters, request and response abstractions, middleware support, blueprints for modular organization, class-based views, a configuration system, cookie handling, error/exception management, signal-based lifecycle hooks, static file serving, WebSocket support, and a test client for testing applications.

## Scope
- ~25 production source files to implement
- ~15 test files to write
- Reproduce core web framework functionality: app, routing, request, response, middleware, blueprints, config, exceptions, views, cookies, signals, static files, websockets, and test client
